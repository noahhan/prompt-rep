# LLM Wiki

This is the local knowledge base for Prompt Vault.

Purpose:
- Store prompt-engineering knowledge.
- Keep raw sources separate from edited notes.
- Help future LLM work update knowledge in a clear way.
- Make the app better at suggesting prompt patterns.

Folder map:

- `00-inbox/`: new raw ideas before cleanup
- `10-sources/`: source summaries and links
- `20-notes/`: working notes
- `30-concepts/`: stable concepts
- `40-playbooks/`: task-specific prompt guides
- `90-meta/`: rules, schema, and changelog

Main rule:

Do not overwrite old knowledge silently. Add a note to `90-meta/changelog.md` when knowledge changes.

Important files:

- `index.md`: map of the wiki
- `log.md`: chronological wiki activity
