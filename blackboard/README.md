# Agent Blackboard

This folder is a shared board for collaboration between agents.

Purpose:

- Share current project context.
- Record ideas from different perspectives.
- Track open tasks.
- Record decisions.
- Avoid duplicated work.

## How agents should use this

1. Read `context.md` first.
2. Read `tasks/open-tasks.md`.
3. Add notes to `notes/`.
4. Add decisions to `decisions/decision-log.md`.
5. Update `tasks/open-tasks.md` when work changes.
6. Keep English simple and clear.

## Folder map

- `context.md`: current project state
- `agents/`: agent role guides
- `notes/`: working notes and perspectives
- `decisions/`: decision records
- `tasks/`: open and completed tasks
- `archive/`: old notes

## Rule

Do not delete another agent's note.
If something is wrong, add a correction note.
