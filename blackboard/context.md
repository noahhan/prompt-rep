# Project Context

Project name: Prompt Vault

Goal:

Build a prompt repository for local users.

Current app:

- Offline web app
- Saves prompts in browser `localStorage`
- Supports Overview card view
- Supports expandable categories and prompt selection in the left sidebar
- Supports categories and tags
- Supports search
- Supports `{placeholder}` variables
- Supports Markdown edit and Preview modes
- Supports safety audit in the Audit tab
- Shows safety score factors, quality factors, checks, suggestions, placeholders, and export warning
- Supports JSON and Markdown import/export
- Supports version history
- Includes 17 starter prompts based on prompt-engineering research
- Merges starter prompts once for existing local users without overwriting saved prompts
- Has LLM Wiki knowledge base
- Has Agent Blackboard for collaboration notes

GitHub:

- Public repository
- URL: https://github.com/noahhan/prompt-rep.git

Important warning:

- This repository is public.
- Never commit `.env`.
- Never commit tokens or secrets.

User preference:

- Use simple and concise English.
- User is not a native English speaker.

Current UI direction:

- Keep the interface clean and calm.
- Avoid too many rounded boxes and heavy icons.
- Overview and Categories are same-level left menu titles.
- Prompt editing area should have broad space.
- Audit is useful, but it should not take permanent large space.
